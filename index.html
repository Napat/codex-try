ไฮไลต์ฟีเจอร์สำคัญ

เข้าใจหลายภาษา

  • รองรับ Python, JavaScript, TypeScript, Go, Rust, C#, Java และอีกมากมาย
  • อ่านบริบทโค้ดในโปรเจ็กต์ได้ครบทั้งไฟล์และโฟลเดอร์

ชำนาญบริบทเชิงลึก

  • จำประวัติการสนทนาเพื่อให้คำตอบต่อเนื่อง
  • แนะนำการปรับปรุงโดยอ้างอิงไฟล์ที่กำลังทำงานอยู่

รีวิวโค้ดอัตโนมัติ

  • วิเคราะห์ diff และ Pull Request พร้อมชี้จุดเสี่ยง
  • เสนอ refactor หรือ test เพิ่มเติมในรูปแบบ bullet

สร้างเทสและเอกสาร

  • เขียน unit / integration test ครบถ้วน
  • ช่วยร่าง README หรือคู่มือประกอบโค้ดได้รวดเร็ว

Workflow เชื่อมต่อ GitHub

  • สั่งเปิด branch, commit, push และ PR ผ่านภาษาไทย
  • ผสานเข้ากับ automation script หรือ GitHub Actions ได้

การเตรียมใช้งาน (Setup)

ขั้นตอน 1

เตรียม API Access

สมัครบัญชีและสร้าง API key ที่ platform.openai.com

export OPENAI_API_KEY="sk-..."
ขั้นตอน 2

ติดตั้ง CLI Toolkit

pip install --upgrade openai
openai --version
openai api keys add default --key "$OPENAI_API_KEY"
ขั้นตอน 3

ตั้งค่า IDE Plugin

ใน VS Code ให้ค้นหา "ChatGPT Codex" หรือปลั๊กอินที่เชื่อม OpenAI API แล้วใส่ API key

ขั้นตอน 4

เชื่อมต่อ GitHub

สร้าง Personal Access Token ที่ GitHub และตั้งค่าตัวแปรสภาพแวดล้อม

export GITHUB_TOKEN="ghp_..."

ตัวอย่างการใช้งาน

ใช้งานผ่าน CLI

openai chat.completions.create \
  --model gpt-4.1-codex \
  --messages '[{"role":"system","content":"คุณคือผู้ช่วยโปรแกรมเมอร์"},
               {"role":"user","content":"ช่วยเขียนฟังก์ชัน merge sort ภาษา Python"}]' \
  --temperature 0.3

ใช้งานผ่าน IDE

  • เปิดไฟล์ใน VS Code แล้วเลือกข้อความที่ต้องการ
  • คลิกขวา > Ask ChatGPT Codex เพื่อขอคำอธิบายหรือเพิ่มเติมโค้ด
  • รับคำตอบและแทรกผลลัพธ์กลับสู่ editor ได้ทันที

รีวิวโค้ด (Code Review)

  • ต่อ API กับ workflow CI/CD เช่น GitHub Actions
  • แนบ diff พร้อมคำสั่งรีวิว โมเดลจะให้ข้อเสนอแนะพร้อมใช้งาน

สั่งสร้างโค้ดผ่านเว็บ

  1. ติดตั้ง web UI ที่รองรับ OpenAI API
  2. ล็อกอิน GitHub OAuth เพื่อเข้าถึง repository
  3. ป้อนคำสั่ง เช่น "สร้าง service จัดการตะกร้าสินค้าในโฟลเดอร์ src/cart"
  4. ระบบจะสร้างโค้ด เปิด Draft PR และพร้อมให้ตรวจสอบ
